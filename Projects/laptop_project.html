Laptop price Prediction

Home Projects Laptop price Prediction

Overview

We are going to build a project to predict the price of laptops. The problem statement is: if a user wants to buy a laptop, our application must be compatible to provide a provisional price of the laptop based on the user's configurations. In addition to the data preparation process and model training in this project we are interested in the integration of MLops tools from tracking to deployment .

Project Organisation

To generate the project structure we used cookiecutter as a solution to organize the project well. cookiecuuter is a tool to create templates for Python projects and use it to generate a project from the command line. Some templates are published on GitHub, etc . This project based on the cookiecutter data science project template.

The main aim of this projects is to build a Mlops pipeline using open-source and free solutions like dagshub, mlflow and git. for that reason we can distinguish 4 parts in this project:

  • Code Versionning : in this part we will use git server hosted on dagshub. this part manage the code of the api that expose differents models to the consumer and webapp application.
  • Model and data versioning : DVC is a solution built to make ML models shareable and reproducible. It is designed to handle large files, data sets, machine learning models, and metrics as well as code. it helps to control machine learning models, data sets and intermediate files version.
  • Model tracking : we use two tracker mlflow and dagshub logger in order to track different metrics and models params
  • Model deployment : we use github actions for ci workflow and store docker images for api and webapp in dockerhub for deployment

Machine learning modeling

We have prepared our data in the previous section. In this part we start the machine learning modeling and find the best algorithm to achieve the best performance. For this we use an automated approach with mlflow as we will see in the rest of this blog

Model training

In this project we will train 4 types of regressions models using scikit-learn:
  • Lineaire regressions (Lr)
  • Random forest (Rf)
  • Ada boost
  • Gradient boosting

```py import importlib import sys import joblib import pandas as pd class train(): def __init__(self) -> None: pass def my_import(self, name): components = name.split('.') module=".".join(components[:-1]) mod = __import__(module) for comp in components[1:]: mod = getattr(mod, comp) return mod def save_model(self, model): model_file = str(type(model).__name__).lower() joblib.dump(model, f'models/{model_file}.pkl') def train(self,model,train_features,train_label): train_data=pd.read_csv(train_features) train_label=pd.read_csv(train_label) model.fit(train_data,train_label) self.save_model(model) arguments=sys.argv print(arguments) train_class=train() my_module = train_class.my_import(arguments[1]) train_class.train(my_module(),arguments[2],arguments[3]) ```
